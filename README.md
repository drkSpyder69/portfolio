## 🚀 Quick Start

### Prerequisites
- **Node.js 18+** - JavaScript runtime
- **npm or bun** - Package manager
- **Git** - Version control

### Installation

```bash
# Clone the repository
git clone https://github.com/KhanSadman/portfolio.git
cd portfolio

# Install dependencies
npm install
# OR using bun (faster)
bun install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Start development server
npm run dev
# OR using bun
bun run dev
```

### Build for Production

```bash
# Build the project
npm run build
# OR using bun
bun run build

# Preview the build
npm run preview
# OR using bun
bun run preview
```

## 🔧 Customization & Template Usage

This portfolio has been designed as an easy-to-use template. All personal information and configurations have been extracted into centralized configuration files so you don't need to hunt down text within components!

### **1. Update Configuration `src/data/config.ts`**
Open the `src/data/config.ts` file and update the following objects with your own information:
- `personalInfo`: Your name, role, email, phone, location, about me text, and resume link.
- `socialLinks`: Links to your GitHub, LinkedIn, Facebook, Instagram, etc.
- `emailJSConfig`: Your EmailJS Service ID, Template ID, and Public Key for the contact form.
- `aboutJourney` & `aboutHighlights`: Sections outlining your backstory and key highlights.
- `skillsData` & `techStack`: Your proficiency in various tools and languages.
- `certificatesData`: Your list of certificates and achievements.

### **2. Update Projects `src/data/projects.ts`**
Open `src/data/projects.ts` to replace the default projects with your actual portfolio work. Follow the existing object structure.

### **3. Replace Assets in `public/`**
- Replace `public/about/profile_pic.jpg` with your own profile image.
- Replace or add new project images inside `public/projects/`.
- Replace or add certificate images/PDFs inside `public/certificates/`.

### **Colors & Themes**
- Modify `tailwind.config.ts` to update the primary, secondary, and accent color schemes.
- Update CSS custom properties in `src/index.css`.
- Customize gradient definitions.

### **Adding New Sections**
1. Create a component in `src/components/`
2. Import in `App.tsx` (or your layout component)
3. Add a navigation link in `src/components/Navigation.tsx`
4. Update anchor links
